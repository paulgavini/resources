Drawing App Help

This page explains all current controls and workflows in the drawing app.

Keep your drawing tab open while using this help page.

Quick Start

  1. Use a stylus to draw on the canvas.
  2. Pick ink color from the three color dots.
  3. Use Eraser for corrections and Undo for step-back actions.
  4. Use New Page when you need more space.
  5. Use Save Project regularly to avoid losing work on refresh.

Drawing Controls (Right Panel)

  • Eraser: switches to erase mode.
  • Undo: undoes the most recent action on the current page (up to 10 states per page).
  • 1cm Grid: toggles faint grey graph grid lines.
  • Full Screen: enter or exit fullscreen view.
  • Clear: clears only the current page.

Project Persistence

  • Save Project: downloads a .json file containing full app state (pages, current page, undo history, color/tool mode, grid state).
  • Load Project: restores the app state from a previously saved project file.
  • Important: refreshing the app clears unsaved in-memory state. Save projects often.

Export and Sharing

  • Export PDF: exports all pages into one PDF file.
  • If 1cm Grid is enabled, the grid is included in exported PDF pages.

Page Management

  • Prev / Next: move between pages.
  • New Page: inserts a new blank page after the current page.
  • Delete Page: removes the current page (with confirmation).
  • Page X of Y: shows your position in the document.

Image Paste and Placement

  • Copy an image, then paste with Ctrl+V (or device paste action).
  • The image appears as a movable overlay.
  • Drag to reposition.
  • Use the bottom-right handle to resize.
  • Place Image commits it to the page.
  • Cancel Image discards the temporary overlay.
  • Keyboard shortcuts in placement mode:
    • Enter: place image
    • Esc: cancel placement

Notes and Limits

  • Stylus is required for drawing; finger touch is ignored for ink strokes.
  • Undo history is capped at 10 states per page.
  • Maximum page count is currently 15 pages.
  • Load Project expects project files created by this app.
