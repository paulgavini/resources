PDF Editor

Drop a PDF, open one, or create a blank PDF

Available features:

  • Draw with blue, green, or red pen colours.
  • Use highlighter mode with yellow, orange, or green.
  • Erase, undo, redo, or clear the current page.
  • Toggle a 1 cm grid overlay.
  • Zoom in/out and fit the page to full width.
  • Navigate pages with previous/next controls and page indicator.
  • Enter full screen and export an annotated PDF.
  • Record MP4 video of full page or visible view.
  • Select audio source and refresh input devices before recording.
