2. Introduction

Write in future tense. The introduction should hook the reader, explain the problem, name the field of mathematics, describe it, and connect that mathematics to the task.

3. Mathematical investigation

Write in present tense. Each investigation cycle should introduce the mathematics, show the working/table/graph, and explain what it demonstrates.

5. Graphs

Use the data table to build one or more graphs. Each graph can select a different X-axis, one or more Y-axis columns, and an optional trendline.

6. Analysis

Write in past tense. Analyse the pattern of results, the reasonableness of the solution, the strengths and limitations of the method, and the improvements that would strengthen the investigation.
